# Minesweeper

Windows Minesweeper clone I made for the final project in AP CSA

![screenshot](screenshot.jpg)

## Features

- Multiple difficulties, including custom difficulty
- Cheats
- Built-in AI player
- Multiple themes
- Fully customizable
- Full keyboard support
